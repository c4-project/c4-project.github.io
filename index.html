The C4 Project

Explosive Compiler Concurrency Testing

What is C4?

C4 is a project to check C compiler concurrency. The goal is to be able to find bugs in the way real-world compilers implement the C11 memory model.

C4 contains several free and open-source (FOSS) projects:

c4f
a fuzzer for litmus tests [github]
c4t
an automatic runner for compiler test campaigns [github]
c4-corpora
a set of litmus tests pre-generated with memalloy [github]
c4-scripts
a set of Bash scripts for automating some c4f and c4t workflows [github]
mutated-llvm
a copy of LLVM instrumented with run-time selectable, concurrency-related, mutations [github]

These projects are designed to be used together with a litmus test runner (such as litmus7) to run automatic tests against compilers on one or more machines.

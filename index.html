The C4 Project

What is C4?

C4 is a project to check C compiler concurrency. The goal is to be able to find bugs in the way real-world compilers implement the C11 memory model.

C4 contains four free and open-source (FOSS) projects:

  • c4f, a fuzzer for litmus tests;
  • c4t, an automatic runner for compiler test campaigns;
  • c4-corpora, a set of Litmus tests pre-generated with memalloy;
  • c4-scripts, a set of Bash scripts for automating some c4f and c4t workflows.

Development on C4 was funded as part of the Interface Reasoning for Interacting Systems project.
